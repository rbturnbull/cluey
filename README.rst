Installation
=======================

The software can be installed using ``pip``

.. code-block:: bash

    pip install cluey

To install the latest version from the repository, you can use this command:

.. code-block:: bash

    pip install git+https://github.com/rbturnbull/cluey.git



.. end-quickstart
